Forúm

Forum simples desenvolvido em Python com o framwork Django! Essa aplicação permite usuários a criar e deletar tópicos, como também comentar nos mesmos.

Simple Chat

Projeto simples de um chat com a intenção testar os conhecimentos de banco de dados. A aplicação Java contém um chat onde se pode mandar mensagens como um certo usuário e alterar o usuário a qualquer momento, a plicação guarda as informações em um banco de dados MySQL

Demo de apresentação: https://youtu.be/IPUrDh095Tc

Movie Manager

App que permite o manego de uma coleção de filmes, permitindo adicionar, deletar e editar informações como título, restrição de idade, gênero e sinópse dos filmes presentes na coleção.
